# haur

*H*elper for the *A*rch *U*ser *R*epository

## Installlation

```bash
git clone https://aur.archlinux.org/haur.git
cd haur
makepkg -si
```

## Usage

Usage is pretty simple.

To install:
```bash
haur <package name>
```

To remove:
```bash
haur -r <package name>
```

If you have any problems, please leave a comment on the AUR page and/or create a GitHub issue
