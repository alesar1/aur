# Plymouth BGRT Lenovo

Plymout theme that shows the password entry below the lenovo firmware logo bringing flicker free bgrt boot to encrypted lenovo devices.

# Pacman hook
In [pacman_hook](https://gitlab.rimikis.de/Leptopoda/plymouth-bgrt-lenovo/-/tree/2e43ac0562ee71bb6c83cc25a93b3519007a35a6/pacman_hook) is an untested pacman hook that could be used to alter the default bgrt theme in place after each update of plymouth. It basically comes down to two sed commands.
