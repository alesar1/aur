# percona-server-mongodb-bin

AUR PKGBUILD for Percona's distribution of MongoDB.

## Installation

Install using your favorite package manager

`trizen -S percona-server-mongodb-bin`

Or, clone the repo and run

`makepkg -si`

Enjoy
